3.7鎚打金工及鍛造

多項選擇題：

請選擇正確答案。

 

  1. 鎚打金工可在金屬________狀態進行成形。

    1.   高溫
    2.   冰點
    3.   冷卻
    4.   液體

  2. 進行鎚打金工時，小心檢查鎚頭有沒有________。

    1.   塗上油漆
    2.   鬆脫
    3.   磁化
    4.   生銹

  3. 常用作鎚打金工的材料是 ________。

    1.   黃銅及紅銅
    2.   鋼
    3.   鋁合金
    4.    生鐵

  4. 鎚打金工的金屬材料在加工前需要進行_________工序使金屬軟化。

    1.   淬火
    2.   回火
    3.   正火
    4.   退火

  5. 下列哪一樣材料不是鎚打金工常用的材料？

    1.   黃銅
    2.   紅銅
    3.   銅
    4.   軟鋼

  6. 下列哪一樣不是鎚打金工的製成品？

    1.   碗
    2.   碟
    3.   鏨
    4.   杯

  7. 下列哪一樣不是鎚打金工的工序？

    1.   退火
    2.   盤形凹陷法
    3.   淬火
    4.   碟形凹陷法

  8. ________是將完成的製品打平至光滑，同時微化製品的晶粒使加工硬化。

    1.   鎚平法
    2.   凸拱法
    3.   盤形凹陷法
    4.   碟形凹陷法

  9. 用哪種工具夾持工件進行鍛造？

    1.   鐵砧
    2.   火鉗
    3.   型砧
    4.   虎鉗

  10. 鍛造是將_________後的金屬敲打成不同形狀。

    1.   拉細
    2.   加壓
    3.   壓扁
    4.   加熱

  11. 常見的鍛造製成品有_______。

    1.   鏨
    2.   碗
    3.   碟
    4.   盆

  12. __________可將方形的鐵枝鍛成不同直徑的圓鐵枝。

    1.   型砧
    2.   上下套鎚
    3.   背鏨刀
    4.   平面鏨
