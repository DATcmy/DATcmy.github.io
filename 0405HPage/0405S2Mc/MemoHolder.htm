便條夾設計

多項選擇題：

  

 

  1. 下列哪一條鋁合金的敍述是不正確的？

    1.   不含鐵的金屬
    2.   表面黑色
    3.   重量較輕
    4.   硬度和軟鋼相近

  2. 下列哪一種不是用鋁合金製造？

    1.   飛機機身
    2.   飲品包裝
    3.   鋁窗
    4.   金工虎鉗

  3. 亞加力膠片加熱後會_________。

    1.   凝固
    2.   變硬
    3.   軟化
    4.   散發出氧氣

  4. 下列哪一種不是用亞加力膠片製造？

    1.   高溫射燈燈罩
    2.   文具
    3.   裝飾品
    4.   廣告招牌

  5. 膠片刀只可用作______切。

    1.   直線
    2.   曲線
    3.   圓形
    4.   折線

  6. 使用棒形電熱器時，只需將熱塑性膠料放於______上，待膠料軟化時，便可進行簡單的屈摺工作。

    1.   電熱棒
    2.   電熱箱
    3.   電熱盒
    4.   電熱板

  7. 棒形電熱器是利用_______使塑料軟化，然後進行屈曲工作。

    1.   熱力
    2.   重量
    3.   壓力
    4.   温度

  8. 亞加力膠片加熱至______C便可屈曲。

    1.   70°
    2.   170°
    3.   270°
    4.   370°

  9. 用______黏合亞加力膠片。

    1.   膠水
    2.   白膠漿
    3.   天拿水
    4.   化膠水

  10. 不可用線鋸床來鋸切______。

    1.   薄木板
    2.   鋼片
    3.   薄木條
    4.   膠片

  11. 一套螺絲攻有_______支。

    1.   一
    2.   二
    3.   三
    4.   四

  12. 製造內螺紋，應使用下列哪種工具？1. 螺絲攻 2. 圓牙鈑 3. 牙板扳手 4. 螺絲攻扳手

    1.   1,3
    2.   2,4
    3.   1,4
    4.   2,3

  13. 開始切削內螺紋時, 應使用

    1.   二度螺絲攻
    2.   斜螺絲攻
    3.   螺絲
    4.   塞狀螺絲攻

  14. 準備製造內螺絲的孔是

    1.   比較螺絲攻的直徑小
    2.   與螺絲攻的直徑相同
    3.   比較螺絲攻的直徑大
    4.   與螺絲攻的直徑無關

  15. 在工場利用________製造內螺紋。

    1.   扳手
    2.   圓牙板
    3.   螺絲
    4.   螺絲攻

  16. 螺絲攻是用_______製造。

    1.   鋁合金
    2.   高碳鋼
    3.   鐵
    4.   銅

  17. 進行最後加深螺紋的工作, 可使用

    1.   二度螺絲攻
    2.   斜螺絲攻
    3.   螺絲
    4.   塞狀螺絲攻

  18. 螺絲攻於使用次序上應為_______。1. 塞狀螺絲攻 2. 斜螺絲攻 3. 二度螺絲攻

    1.   1,2,3
    2.   2,1,3
    3.   3,1,2
    4.   2,3,1

  19. ______多用來切斷較幼的金屬桿或電線。

    1.   組合鉗
    2.   剪鉗
    3.   平咀鉗
    4.   電線鉗

  20. _____是一種具備多種用途的鉗，可用作夾持工件，將金屬材料屈摺成形及剪斷幼鐵線。

    1.   組合鉗
    2.   剪鉗
    3.   平咀鉗
    4.   電線鉗

  21. _________可用來夾持薄片或較細小的工件，屈摺成直角。

    1.   組合鉗
    2.   剪鉗
    3.   平咀鉗
    4.   電線鉗
