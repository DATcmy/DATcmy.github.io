3.2劃線及夾持工具

多項選擇題：

請選擇正碓答案。

 

  1. 平板的用途：

    1.   測量工件光滑度
    2.   測試物料表面平坦度
    3.   測量工件厚度
    4.   測量工件長度

  2. 樑規的用途：

    1.   繪畫厚度較長工件
    2.   繪畫硬度較大工件
    3.   繪畫直徑較大的圓形
    4.   繪畫直徑較小的弧線

  3. 下列哪一樣不是V型槽座與鉗的用途？

    1.   夾持圓柱狀的工件
    2.   進行畫線
    3.   定圓心
    4.   切削V型坑槽

  4. 下列哪種工具用作夾持圓形工件在鑽床上鑽孔？

    1.    V型槽座與鉗
    2.   手虎鉗與平板
    3.   大力鉗與平板
    4.   角板與劃針座

  5. 下列哪種工具用作工件劃水平線？

    1.   V型槽座與鉗
    2.   手虎鉗與平板
    3.   大力鉗與平板
    4.   角板與劃針座

  6. 下列哪一樣的敘述不屬於樑規？

    1.   可繪畫直徑較大的圓形
    2.   有一個樑規頭
    3.   有一枝長規桿
    4.   用鋼頂尖進行繪畫工作

  7. 平鈑能在不平坦的表面工作，所以設計以_______隻鈑作為支撐。

    1.   2
    2.   3
    3.   4
    4.   5

  8. 平鈑是以_______鑄成的測試工具。

    1.   軟鋼
    2.   鋁合金
    3.   不銹鋼
    4.   生鐵

  9. 使用_______固定圓柱狀工件進行繪畫圓心。

    1.   角鈑
    2.   平鈑
    3.   Ｖ型槽座與鉗
    4.   樑規

  10. 在平鈑表面塗上________及加上木蓋保護。

    1.   泠卻液
    2.   防銹油
    3.   清水
    4.   塗改液

  11. 可使用_______在較大面積的扁料上繪畫與扁料邊平行的線條。

    1.   樑規
    2.   角鈑
    3.   劃線針座
    4.   Ｖ型槽座

  12. 角板的用途

    1.   協助畫線工作
    2.   協助夾持圓形工件
    3.   協助角度測試
    4.   協助角度描繪

  13. V型槽座與鉗的用途

    1.   夾持V型工件
    2.   夾持圓形工件
    3.   夾持三角形工件
    4.   夾持不規則工件

  14. 劃線針座的用途是

    1.   繪劃平行線
    2.   繪劃曲線
    3.   把多點連接
    4.   繪劃斜線
